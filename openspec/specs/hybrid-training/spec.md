# Hybrid Training — Reference Specification

> Based on The Hybrid Coach channel methodology.
> Sources: youtube.com/@TheHybridCoach-e5n, youtube.com/@Rhi-TheHybridCoach

## Philosophy

Hybrid training is the systematic combination of **strength training** and **endurance/cardio training** within a single program. The goal is to develop both force production (heavy compound lifts) and aerobic/anaerobic capacity (running, rowing, ski erg) without compromising adaptation in either domain.

The Hybrid Coach approach emphasizes:

1. **Concurrent programming** — strength and endurance trained in the same week, often the same session
2. **HYROX readiness** — the 8km + 8 station race format drives session design
3. **Progressive overload in both domains** — lift heavier AND run faster
4. **Recovery management** — balancing CNS load from heavy lifts with systemic load from cardio
5. **Compound-first strength** — squat, deadlift, bench, overhead press, row as the foundation
6. **Functional stations** — sled push/pull, farmers carry, wall balls, sandbag lunges, rower, ski erg

## The HYROX Race Format

| Component | Details |
|-----------|---------|
| Total distance | 8 km (8 × 1 km runs) |
| Stations | 8 functional fitness stations (1 between each km) |
| Station 1 | SkiErg (1000 m) |
| Station 2 | Sled Push (50 m) |
| Station 3 | Sled Pull (50 m) |
| Station 4 | Burpee Broad Jumps (80 m) |
| Station 5 | Rowing Machine (1000 m) |
| Station 6 | Farmers Carry (200 m) |
| Station 7 | Sandbag Lunges (100 m) |
| Station 8 | Wall Balls (75-100 reps) |

## Hybrid Training Principles

### 1. Strength Blocks (3-4 weeks)

Focus on max strength in the main lifts. Lower volume, higher intensity (RPE 7-9).

| Week | Focus | Sets × Reps | Intensity |
|------|-------|-------------|-----------|
| 1 | Accumulation | 4 × 8 | RPE 7 |
| 2 | Intensification | 5 × 5 | RPE 8 |
| 3 | Peak | 4 × 3 | RPE 9 |
| 4 | Deload | 3 × 5 | RPE 5-6 |

### 2. Hybrid Blocks (4-6 weeks)

Strength work followed by a conditioning piece in the same session. 

Template: **Strength work** (20-30 min) → **Conditioning** (10-20 min)

### 3. Endurance Blocks

Run-focused with maintenance strength. Lower intensity strength, higher run volume.

### 4. HYROX-Specific Blocks

Race-practice sessions mimicking the HYROX format.

### Key Exercise Categories

| Category | Examples | Purpose |
|----------|----------|---------|
| Main lifts | Squat, Deadlift, Bench, OHP, Barbell Row | Strength foundation |
| Strength accessories | RDL, Bulgarian Split Squat, Pull Ups, Dips | Muscle balance, injury prevention |
| Power | Power Clean, Hang Clean, Push Jerk | Rate of force development |
| Sled work | Sled Push, Sled Pull | Leg drive, conditioning |
| Carry | Farmers Carry, Suitcase Carry, Waiter Walk | Grip, core, work capacity |
| Monosteady cardio | Run, Rower, SkiErg, Assault Bike | Aerobic base |
| HYROX stations | Wall Balls, Burpee Broad Jumps, Sandbag Lunges, Farmers Carry | Race-specific |
| Intervals | 400m repeats, 1km repeats, Tabata row | Speed, lactate tolerance |

## Session Templates

### Strength + Run (Hybrid Default)

```
A1. Main Lift: 4 × 6 @ RPE 7-8
A2. Accessory: 3 × 10 @ RPE 7
B. Run: 3 km @ Zone 3-4 (tempo)
```

### HYROX Race Simulation

```
A. SkiErg: 500 m @ race pace → rest 1 min → repeat 3×
B. Run: 1 km @ 5k pace
C. Wall Balls: 50 reps
D. Run: 1 km @ 5k pace
E. Sled Push/Pull: 25 m each
F. Repeat B-E once more
```

### Strength Focus (Heavy Day)

```
A. Main Lift: 5 × 3 @ RPE 8-9
B. Second Lift: 4 × 6 @ RPE 7-8
C. Accessories: 3 × 10-12
D. Core: 3 rounds
```

### Conditioning

```
A. Rower: 500 m @ 1:50 split → 20 air squats → 10 burpees
   × 5 rounds for time
B. Zone 2 run: 20 min @ conversational pace
```

### EMOM Hybrid

```
EMOM 12:
Min 1: 10 Wall Balls + 5 Burpees
Min 2: 15 Cal Row
Min 3: 8 Deadlifts @ 60%
Min 4: Rest
```

## Periodization Example (12-week HYROX prep)

| Weeks | Block | Focus |
|-------|-------|-------|
| 1-4 | Strength Foundation | Main lifts 4×8→5×5, easy runs 2×/week |
| 5-8 | Hybrid Build | Strength + conditioning in same session, HYROX stations, tempo runs |
| 9-11 | Race Specific | Full HYROX simulations, wall ball volume, sled work |
| 12 | Taper/Deload | Reduced volume, race pace practice |
