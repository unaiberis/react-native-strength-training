# The Hybrid Project · v0.1

Primera versión base del proyecto Expo / React Native.

## Incluye

- Pantalla inicial de acceso.
- Home del atleta con calendario y entrenamiento del día.
- Placeholder del panel entrenador.
- Sistema visual oscuro: negro, gris titanio, blanco roto.
- Componentes reutilizables iniciales.
- Datos mock para probar navegación e interfaz.

## Instalación

```bash
npm install
npm run start
```

## Estructura

```txt
app/
  index.tsx
  (auth)/
  (athlete)/
  (coach)/
components/
constants/
data/
types/
```

## Nota

Los logos e imágenes reales deben añadirse en `assets/images/`.
En esta versión el logo está representado por texto para evitar errores de assets.
